# 🧠 CogniSync AI

**Autonomous Multi-Agent Architecture for Competitive Examination Mastery**

CogniSync is a next-generation study platform that transforms raw syllabus data into structured, high-intensity learning paths. By leveraging a Multi-Agent system (Tutor & Planner Agents) powered by Google Gemini, CogniSync acts as an academic supervisor for students preparing for high-stakes engineering exams.

## 🚀 Features
- **Local PDF Knowledge Base:** Upload your own syllabus or study materials securely.
- **Tutor Agent:** Dynamically generates conceptual problems and detailed mathematical derivations based on your specific curriculum.
- **Planner Agent:** Creates personalized, high-intensity weekly revision countdown schedules.
- **Autonomous Routing:** Intelligent supervisor delegates tasks to specialized AI agents.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Brain:** Google Gemini API
- **Logic:** Python (Multi-Agent Architecture)
- **Data:** PyPDF2 (Local PDF Processing)
